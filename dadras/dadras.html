DADRAS ATTRACTOR

The Dadras or Dadras-Momemi Attractor looks like a complex wave trapped into three dimensions. It can be used to describe a Wang 3-scroll chaotic system. The most interesting part about the Dadras attractor though is that the system can accept random values within a range as parameters and generate interesting shapes with chaotic behavior.

and a system of three ordinary differential equations describing the appropriate attractor are- $\left\{ \begin{eqnarray*} \frac{dx}{dt}&=& y - a x +b y z\\ \frac{dy}{dt}&=& c y - x z +z\\ \frac{dz}{dt}&=& d x y - e z \end{eqnarray*}\right. $

When a = 3.0, b = 2.7, c = 1.7, d = 2.0 and e = 9.0, a solution curve of this system has the shape of a complex wave and the position of particles will follow a similar path.
